# MiniRisk PPO Agent – Setup Guide

This project trains and evaluates a reinforcement learning agent that learns to play a simplified Risk-style strategy game using Maskable PPO.

---

## Setup
### 1. Install dependencies
    pip install -r requirements.txt

### 2. Train the PPO Agent
    python train.py

### 3. Evaluate the Agents
    python evaluate.py

### 4. Watch the Agent Play
    python visualize.py
